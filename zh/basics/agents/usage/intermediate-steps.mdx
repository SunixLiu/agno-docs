---
title: 智能体中间步骤流式传输
---

本示例演示了如何在智能体执行期间流式传输中间步骤，从而提供对工具调用和执行事件的可见性。

<Steps>

  <Step title="创建一个Python文件">
    ```bash
    touch intermediate_steps.py
    ```
  </Step>


  <Step title="添加以下代码到你的Python文件">
    ```python intermediate_steps.py
    from typing import Iterator

    from agno.agent import Agent, RunOutputEvent
    from agno.models.openai import OpenAIChat
    from agno.tools.duckduckgo import DuckDuckGoTools
    from rich.pretty import pprint

    agent = Agent(
        model=OpenAIChat(id="gpt-5-mini"),
        tools=[DuckDuckGoTools(stock_price=True)],
        markdown=True,
    )

    run_stream: Iterator[RunOutputEvent] = agent.run(
        "What is the stock price of NVDA", stream=True, stream_events=True
    )
    for chunk in run_stream:
        pprint(chunk.to_dict())
        print("---" * 20)
    ```
  </Step>

  <Snippet file="create-venv-step.mdx" />

  <Step title="安装依赖">
    ```bash
    pip install -U agno openai ddgs rich
    ```
  </Step>

  <Step title="设置 OpenAI API key">

    <CodeGroup>

    ```bash Mac/Linux
      export OPENAI_API_KEY="your_openai_api_key_here"
    ```

    ```bash Windows
      $Env:OPENAI_API_KEY="your_openai_api_key_here"
    ```
    </CodeGroup>
  </Step>

  <Step title="运行智能体">
    <CodeGroup>
    ```bash Mac
    python intermediate_steps.py
    ```

    ```bash Windows
    python intermediate_steps.py
    ```
    </CodeGroup>
  </Step>

</Steps>
