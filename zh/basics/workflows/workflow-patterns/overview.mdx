构建确定性的、可用于生产的工流，以可预测的执行模式编排智能体、团队和函数。本综合指南涵盖了所有工作流类型，从简单的顺序流程到具有并行执行和动态路由的复杂分支逻辑。
与自由形式的智能体交互不同，这些模式通过结构化自动化提供了一致且可重复的结果，非常适合生产系统。

## 构建模块

Agno 工作流的核心构建模块是：

| Component     | 目的 |
| ------------- | --- |
| **Step**      | 基础执行单元 |
| **Agent**     | 具有特定角色的 AI 助手 |
| **Team**      | 协同工作的智能体组 |
| **Function**  | 自定义 Python 逻辑 |
| **Parallel**  | 并发执行 |
| **Condition** | 条件执行 |
| **Loop**      | 迭代执行 |
| **Router**    | 动态路由 |


Agno 工作流支持多种执行模式，可以组合这些模式来构建复杂的自动化系统。
每种模式都服务于特定的用例，并且可以组合在一起以实现复杂的工作流。
